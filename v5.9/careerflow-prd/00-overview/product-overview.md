# CareerFlow PRD v5.9（MVP）
# 产品概览

## 用户痛点
- 不会写简历
- 无法量化成果
- 每次投递重复劳动
- 没有长期积累

## 市场问题
- 模板工具：只改格式
- AI工具：容易造假

## 核心方案
构建“成就库”：
- 提取真实经历
- AI优化表达
- 按JD动态组合

## 核心定位
AI驱动的“成就库 + JD定制简历”系统

## 核心逻辑
先沉淀用户真实成就 → 再按JD智能组装简历

## 两条路径
- F1：上传旧简历 → 自动提取成就
- F2：Notion接入 → 持续积累成就

## MVP关键优化（v5.9）
- 上传后直接进入工作台
- 简历语言切换移入工具栏
- 照片功能极简（开关）
- 支付路径 ≤ 3步

## 系统核心模块
- 成就库（核心数据层）
- 工作台（核心交互层）
- AI美化系统（三档）
- JD匹配系统
- 支付系统

## 技术架构
- Next.js + Supabase + AI模型（Qwen + Claude）
