## 付费流程（en_paid市场）

```
点击导出 → 弹窗显示价格 → 选择套餐 → 支付 → 完成
   (1步)      (2步)         (3步)
```

## 组件规范

### 1. 格式选择
- 单选按钮组
- 默认选中: PDF
- 图标 + 文字说明

### 2. 付费墙（仅en_paid）
- 触发条件: profiles.payment_market = 'en_paid' AND 未订阅
- 价格显示: 从 paywall_settings 表读取
- 支付按钮: 主要按钮样式

### 3. 支付集成
- Stripe Checkout
- 成功后: 更新订阅状态 + 自动下载
- 失败: 错误提示 + 重试

## 状态处理

- 生成中: 进度条
- 成功: 自动下载 + 成功提示
- 失败: 错误信息 + 重试按钮
