# 版本历史侧边栏设计

## 触发方式
- 点击 Workspace 工具栏的 [📜 历史] 按钮
- 从右侧滑入侧边栏

## 布局结构

```

│                                     │ │ 版本历史        [✕] ││
│                                     │ ├─────────────────────┤│
```

## 组件规范

### 侧边栏容器
- 宽度: 400px
- 背景: 白色 / 深色模式
- 阴影: -4px 0 12px rgba(0,0,0,0.1)
- 动画: 从右侧滑入 (300ms ease-out)

### 版本卡片
- 高度: 自适应
- 边距: 12px
- 圆角: 8px
- 边框: 1px solid #E5E7EB
- Hover: 背景色 #F9FAFB

### 按钮
- [查看]: 次要按钮，打开预览
- [恢复]: 主要按钮，确认后恢复到该版本
- [✕]: 关闭侧边栏

## 交互逻辑

1. **查看版本**: 在右侧预览区显示该版本内容（只读）
2. **恢复版本**: 弹出确认对话框 → 恢复 → 关闭侧边栏
3. **搜索**: 实时过滤版本列表
4. **分组**: 按日期分组（今天、昨天、本周、更早）

## 数据来源
- 表: `resume_versions`
- 字段: `snapshot_label`, `created_at`, `editor_json`
