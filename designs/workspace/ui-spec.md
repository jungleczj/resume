# 同屏工作台界面设计规范

## 页面目标
- 左右分屏：JD粘贴区+成就库 | 简历实时预览
- 支持拖拽成就到简历
- 实时编辑简历内容
- 照片上传与开关控制

## 核心组件

### 1. 顶部工具栏
- 高度: 56px
- 背景: 白色/深色
- 按钮间距: 12px
- 照片开关: CN市场默认开，EN市场默认关
- [📜 历史] 按钮: 点击打开版本历史侧边栏

### 1.1 版本历史侧边栏
- 宽度: 400px
- 位置: 从右侧滑入，覆盖在预览区上方
- 动画: 300ms ease-out
- 背景: 白色，阴影 -4px 0 12px rgba(0,0,0,0.1)
- 关闭: 点击 [✕] 或点击遮罩层

**版本卡片**:
- 显示: 版本号、时间、JD 标签（如有）
- 操作: [查看] 在预览区显示该版本（只读）
- 操作: [恢复] 弹出确认对话框后恢复
- 分组: 按日期分组（今天、昨天、本周、更早）

### 2. 左侧面板 (可调整宽度 30%-50%)
**JD粘贴区 (上部 40%)**
- Textarea: 自适应高度
- 字数限制: 5000字
- 超长自动截断
- 生成按钮: 主要按钮样式

**成就库 (下部 60%)**
- Tab切换: 成就库 / 草稿
- 搜索框: 实时过滤
- 成就卡片: 拖拽手柄 + 插入按钮
- 颜色标识: 🟢Tier1 🟡Tier2 🔴Tier3

### 3. 右侧简历预览
- TipTap 富文本编辑器
- 照片区: 150×200px, 右上角
- 可编辑: 所有文本内容
- 不可复制: 禁用选择和复制功能

## 交互规范

### 拖拽成就
- 桌面端: 从左下拖入右侧编辑器
- 移动端: 点击「插入」按钮
- 插入位置: 当前光标位置或工作经历末尾

### 照片功能
- 上传: 点击照片区上传
- 开关: 工具栏照片按钮切换显示/隐藏
- 格式: JPG/PNG, 最大2MB
- 尺寸: 自动裁剪为 3:4 比例

### 语言切换
- 下拉菜单: 中文 / English
- 切换后: 重新生成简历内容
- 保存偏好: 存入 profiles.resume_lang_preference

## 响应式设计

**桌面端 (≥1024px)**
- 左右分屏布局
- 支持拖拽调整宽度

**平板端 (768px-1023px)**
- 左右分屏，固定比例
- 成就库可折叠

**移动端 (<768px)**
- 上下堆叠布局
- Tab切换: JD / 成就库 / 预览
